Grundläggande element

Meta

html - huvudelement som innehåller hela htmläfiles element

Head - beskriver dokumentet till datorn

Style - Hanterar stilen på alla element på hemsidan

Layout

header - sidhuvud innehåller information om sidan. Kan ev innehålla rubrik

main - innehållet på sidan innehåller flera sections

title - definerar titlen på hemsidan, dels visar den titeln på fliken

footer - Är botten på hemsidan och har alla element som är där

Text och rubriker

h1,h2,h3,h4,h5,h6 - Olika headers, det hjälper med organisering

p - paragrafer som man kan skriva text i

Stilar

Display - Den styr gyr element arrangeras med olika layout metoder

Font-family - Bestämmer vilket tecksnitt texten ska ha på hemsidan

color - Utgör färgen på ett ellement eller text

background-color - Bestämmer backgrund färgen
